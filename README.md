# UNP_PROJECT_BI_WITH_POWER_BI_NETFLIX_DATA_ANALYSIS

I completed a Netflix Data Analysis project in collaboration with UNP using the given data. For this project we were divided in groups and we were given a Netflix's userbase data. The main objective of this project is to analyze Netflix’s userbase and understand subscription patterns, revenue contribution, and user behavior across different demographics such as country, gender, age, and device usage. For this analysis, I have used Microsoft Power BI as the visualization and analytics tool. The report highlights key performance indicators such as Total Revenue (31K) and Total Subscriptions (2500), providing a quick overview of overall platform performance. It includes detailed demographic analysis based on gender, age group, and country to understand customer distribution and behavior patterns. Revenue performance is analyzed country-wise to identify high-performing markets such as the United States and Spain, while also comparing contributions from other regions. Device-wise analysis showcases how users access Netflix through laptops, tablets, smartphones, and smart TVs, helping evaluate platform accessibility and user engagement. A gender-based revenue comparison provides insights into balanced customer contribution across regions. Additionally, the project includes time-series analysis to track monthly revenue trends, identifying peak growth periods and decline phases for strategic planning. Interactive slicers enable dynamic filtering by country, device, gender, and subscription type, enhancing user-driven analysis. Overall, this project demonstrates strong skills in DAX calculations, dashboard design, and business intelligence using Power BI.
